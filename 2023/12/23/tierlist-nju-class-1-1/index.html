NJU Classes Tierlist - Freshman Term 1

这里是XGN的评价时间！在南京大学即将度过一个学期，是时候评价一下所选的各位课程、老师、助教了！

TL;DR

思想政治

课程内容：C - 可以说是非常无聊

老师zw：A - 老师生动有趣 十分幽默

英语听说

课程内容：C - 十分弱智，无聊至极，想睡觉，偶尔有点有意思的听力

老师ww：A - 和蔼温柔 亦师亦友

线性代数（LA）

课程内容：D - 定义+性质+证明+repeat

老师ccx：B - 上课讲的想昏睡

助教：S - 批改十分认真及时！

微积分（Calculus）

课程内容：C - 一般般吧

老师xyy：B - 会补充很多内容，但是上课十分想昏睡，而且总是喜欢占满课堂时间……

助教：D - 作业批改只有一个勾，无论对错。

英语读写

课程内容：C - 就是普通的英语课，但是教材错误十分多

老师wxh：A - 好

信息与计算科学导论（ICSI）

课程内容：S - 神

老师zs：S - 神

助教lsa：C - 作业批改速度慢，吹毛求疵

助教zty：B - 态度高傲

游泳初级

课程内容：B - 有点慢

老师jsq：B - 还行吧

信息与计算科学导论实验（ICSIE）

课程内容：A - 比较快，可以向普通班级的同学炫耀

老师tw：C - 上课没人听吧

助教zty：B - 同上

助教zzx：S - 最会循循善诱、出题的人

助教wsr：E - 准备的题目难度太高、无聊、知识点重复、出完题就跑路，不会辅导教育出题可以不搞的

《控制论》阅读

老师xxl：A - 事少就行

《科学研究的艺术》阅读

老师sz：A - 事特别少

青年科学之光：程序设计语言珠玑（PLP）

课程内容：A - 就没几次课，内容有收获

老师lhj：A - 耐心、严格，没什么表情
